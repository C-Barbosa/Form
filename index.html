Estrutura Básica e HTML

(Voltar)

HTML: (Hypertext Markup Language) é a linguagem padrão para a criação e estruturação de páginas web. A estrutura básica de um documento HTML consiste em elementos que fornecem significado e formatação para o conteúdo da página.

Elemento HTML: Um elemento HTML é composto por tags que envolvem o conteúdo e fornecem significado e estruturação. As tags geralmente vêm em pares, uma de abertura e outra de fechamento . O conteúdo vai entre essas tags.

Estrutura Básica: A estrutura básica de um documento HTML inclui as seguintes partes: html, head, tittle e body.

Atributos de Tags

(Voltar)

Os atributos de tags em HTML são usados para fornecer informações adicionais sobre os elementos HTML, ajudando a controlar seu comportamento e aparência.

Atributos Globais: Aplicáveis a qualquer elemento HTML, como id, class, style, title, lang, dir, etc.

Atributos Específicos: apenas a elementos específicos, como src em <.img>, href em <.a>, type em <.input>, etc.

Listas ordenadas e não ordenadas

(Voltar)

As listas ordenadas e não ordenadas são recursos fundamentais em HTML para organizar e apresentar informações de forma estruturada.

Listas Não Ordenadas <.ul>:

As listas não ordenadas são usadas para apresentar itens em uma sequência que não requer ordem específica e são definidas pelo elemento <.ul> (unordered list) e cada item da lista é definido pelo elemento <.li> (list item). A apresentação padrão é com marcadores, geralmente pontos, antes de cada item.

Listas Ordenadas <.ol>:

As listas ordenadas são usadas para apresentar itens em uma sequência específica, com uma ordem numérica ou alfabética, são definidas pelo elemento <.ol>(orldered ist)e cada item da lista é definido pelo elemento <.li> (list item). A apresentação padrão é com números ou letras antes de cada item.
